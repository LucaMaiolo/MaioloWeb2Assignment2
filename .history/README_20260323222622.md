## Rest API Design Table

|     Purpose          |     Method    |     Endpoint        |     Controller Fn                   |     Model Fn           |
|----------------------|---------------|---------------------|-------------------------------------|------------------------|
|     Home page        |     GET       |     /               |     homeController: showHome        |     —                  |
|     Invalid URL      |     ANY       |     *               |     errorController: handleError    |     —                  |
|     Create a job     |     POST      |     /jobs           |     jobController: addJob           |     addJob()           |
|     List all jobs    |     GET       |     /jobs           |     jobController: getAllJobs       |     getAllJobs()       |
|     Show one job     |     GET       |     /jobs/:title    |     jobController: getJob           |     getJobByTitle()    |
|     Update a job     |     PUT       |     /jobs/:title    |     jobController: updateJob        |     updateJob()        |
|     Delete a job     |     DELETE    |     /jobs/:title    |     jobController: deleteJob        |     deleteJob()        |

## ENDPOINTS

GET  /
  → Home page welcome message

POST /jobs
  → Create a new job
  Body: {
    "title": "Fix leaky faucet",
    "description": "Kitchen faucet drips constantly.",
    "location": "Montreal, QC",
    "budget": 80
  }

GET  /jobs
  → List all jobs
  Optional: /jobs?status=open
  Optional: /jobs?status=in-progress
  Optional: /jobs?status=completed

GET  /jobs/:title
  → Get one job by title
  Example: /jobs/Fix leaky faucet

PUT  /jobs/:title
  → Update budget and status of a job
  Example: /jobs/Fix leaky faucet
  Body: {
    "budget": 120,
    "status": "in-progress"
  }

DELETE /jobs/:title
  → Delete a job by title
  Example: /jobs/Fix leaky faucet
